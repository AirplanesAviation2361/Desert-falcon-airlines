# Desert Falcon Airways – GeoFS Livery Pack

Desert Falcon Airways is a virtual airline created for the GeoFS Livery Selector.
This pack adds custom liveries for selected aircraft in GeoFS.

# Airline Information

Name: Desert Falcon Airways
Theme: Middle Eastern style airline
Primary Color: Gold (#C9A227)
Background Color: Dark Blue (#0B1F3A)

# Fleet

The current Desert Falcon Airways fleet includes:
	•	Airbus A321neo – Regional and short-haul routes
	•	Boeing 757-300 – Medium-haul routes and classic aircraft
	•	Airbus A330-900neo – Long-haul flagship aircraft

# Installation (GeoFS Livery Selector)
1.	Open GeoFS.
2.	 2.	Open the Livery Selector.
  
3.	Click + Virtual Airline.
  
4.	Paste the Raw link of airline.json from this repository.
  
5.	Press Confirm.

The Desert Falcon Airways liveries will now appear in the aircraft livery list.

# Repository Contents
•	airline.json – Airline configuration for the GeoFS Livery Selector
•	textures/ – Aircraft livery textures used in the pack
